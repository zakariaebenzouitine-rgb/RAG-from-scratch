# RAG From Scratch

This project is a hands-on implementation of a Retrieval-Augmented Generation (RAG) pipeline.

## Goals
- Understand RAG concepts end-to-end
- Build document ingestion, embedding, retrieval, and generation
- Use modern LLM tooling (vector databases, embeddings, prompt chaining)

## Tech Stack (planned)
- Python
- LangChain or custom pipeline
- Vector database (FAISS / Chroma)
- OpenAI / local LLMs
